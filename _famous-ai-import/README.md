# HyriseCrown Medical Procedures App

A comprehensive medical procedures platform for healthcare professionals worldwide.

## Features

- **80+ Medical Procedures** across 8 specialties
- **Evidence-Based Protocols** from top hospitals (Johns Hopkins, NHS UK, Mayo Clinic, etc.)
- **AI-Powered Chatbot** for instant assistance
- **Advanced Search & Filtering** by specialty, complexity, duration
- **Step-by-Step Instructions** with equipment checklists
- **Role-Based Access** for doctors, surgeons, nurses, and caregivers
- **Downloadable Resources** and clinical guidelines
- **Saved Procedures** for quick reference

## Specialties Covered

1. General Surgery
2. Dermatology
3. Orthopedics
4. ENT (Otolaryngology)
5. Ophthalmology
6. Cardiology
7. Gastroenterology
8. Urology

## Getting Started

```bash
npm install
npx expo start
```

## Tech Stack

- React Native with Expo
- Expo Router for navigation
- TypeScript
- Context API for state management
